# PIN DOCS Document Search Website

## Deskripsyon
Maong website alang sa sayun nga pagpangita sa imong mga dokumento sa PIN DOCS folder. Gidesinyo para makita ug makit-an dayon ang mga PDF files sa CDU, Test Head, ADB, ug Cabinet categories.

## Mga Features
- **Quick Search**: Pangita ang mga dokumento gamit ang filename o keywords
- **Category Filter**: Filter segun sa category (CDU, TEST HEAD, ADB, CABINET)
- **Slot Filter**: Filter para sa 12 SLOT o 24 SLOT nga mga dokumento
- **Real-time Results**: Instant search results samtang nagsulat ka
- **Document Preview**: Open ang PDF files direkta gikan sa website
- **Download**: Download ang mga dokumento
- **Keyboard Shortcuts**: 
  - `Ctrl+K` para ma-focus sa search box
  - `Escape` para ma-clear ang search

## Giunsa Pag-gamit
1. Open ang `index.html` file sa imong browser
2. Type ang imong gipangita sa search box
3. Gamita ang filters para ma-refine ang search
4. Click sa "Open" para ma-view ang PDF
5. Click sa "Download" para ma-download ang file

## Browser Compatibility
- Chrome/Edge (recommended)
- Firefox
- Safari

## Mga Notes
- Ang website nag-work offline (walay internet needed)
- Ang mga PDF files kay direktang ma-open gikan sa imong computer
- Kung dili ma-open ang file, check ang file path sa error message

## Troubleshooting
**Problem**: Dili ma-open ang PDF files
**Solution**: Gamit ang Chrome o Edge browser, o check kung ang PDF reader kay installed sa imong computer.

**Problem**: Walay makit-an nga results
**Solution**: Check ang spelling sa imong search, o clear ang filters gamit ang "Clear" button.
